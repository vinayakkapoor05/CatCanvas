CatCanvas

About:
CatCanvas is a Chrome extension that enhances your Canvas experience with AI-powered tools for smart course management, planning, and deadline syncing.

Built by:
Edbert Handjaja, Harry White, Neo Trovela-Villamiel, Romir Mohan, Vicheda Narith, Vinayak Kapoor

Usage Tips:

  • Re-sync: Run this while on the Canvas Dashboard.
  • Get Course History: Click “Courses” on the Canvas sidebar before using this.

Instructions:

  • Re-sync: Must be on the Canvas dashboard page.
  • Get Course History: You must have clicked “Courses” on the Canvas sidebar first.
